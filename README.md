# HR Attrition Analysis Dashboard

To analyze employee attrition trends and workforce demographics to help the HR department understand key factors influencing employee turnover and develop effective retention strategies. 

## 📊 Project Overview
This is a static HR dataset project aimed at understanding employee attrition patterns.

- Tools Used: Python, Power BI
- Dataset: HR Employee Data from Kaggle
  (https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)
- Objective: Identify and visualize key drivers of attrition. 

## 🧰 Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI (Dashboard visualization)
- Excel/CSV (Data storage)
- Ms Word (Report writing)

## 📈 Key Insights
- Overall attrition rate: 16%
- Major factors affecting attrition:
  - Distance from Home
  - Number of Companies Worked
  - Monthly Rate
  - Performance Rating

## 🧠 Recommendations
- Offer hybrid work for employees far from office
- Improve retention for job-hopping candidates
- Benchmark salary structure
- Provide growth and recognition programs
